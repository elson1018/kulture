# Kulture

This project is a web application with a React frontend and a Java (Maven) backend.

## Project Structure

- **frontend/**: React application (Vite).
- **backend/**: Java backend services (MappingServlets).

## Getting Started

### Frontend
1. Navigate to `frontend/`:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Backend
1. Navigate to `backend/MappingServlets/`:
   ```bash
   cd backend/MappingServlets
   ```
2. Use Maven to build and run the project:
   ```bash
   mvn clean package cargo:run
   ```
